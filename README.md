# qw
QuickViewer  
Simplify large commands, using qw 

Installing:  
python3 setup_qw.py  
Then set bindings you want step by step (bind-command)  

Using:  
qw {bind}  
qw --add-bind  
  
  
Bugs:
set fileformat to "unix" if you see "bash: /usr/bin/qw: /usr/bin/python3^M: bad interpreter: No such file or directory"
